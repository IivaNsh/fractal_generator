# simple cell automata

#### only have Linux support

to make app:
```
  git clone https://github.com/IivaNsh/cell_automata
  cd cell_automata/build
  cmake ..
  make
```

launch app:
```
  ./app
```

shortcuts:

```
 RMB - add pixel(no hold)
"q" - exit
"d" - toggle menu
"s" - start/stop animation
"r" - reset texture and stop animation
```
